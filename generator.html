Notes importantes :

• Les noms de colonnes doivent rester exactement les mêmes : Nom, Prénom, Classe, DateNaissance, LieuNaissance, Sexe, Matricule

• Pour les photos : nommez les fichiers Nom_Prenom.jpg (ex: KOSSI_Aménam.jpg) ou Matricule.jpg (ex: CP1-001.jpg)

• Le format de date recommandé est JJ/MM/AAAA

• Le sexe peut être "M" pour Masculin ou "F" pour Féminin
